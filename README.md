# Plantilla Just the Docs (para clase)

Esta es una **plantilla de documentación** basada en **Just the Docs** (Jekyll) para que:

- descarguen/clonen un repositorio,
- editen páginas Markdown,
- publiquen en **GitHub Pages**.

Incluye header/footer personalizados y carga CSS adicional desde `head_custom.html` y un footer en `footer_custom.html` con créditos/licencia.

---

1. Crea un repositorio, copiando este repositorio con **Fork**.
2. Cambia `_config.yml`:

```yml
url: "https://TU_USUARIO.github.io"
baseurl: "/NOMBRE_DE_TU_REPO"
```

3. Activa GitHub Pages en tu repo: **Settings → Pages → Deploy from a branch → main / root**.


4. Editar contenido

- La home está en `index.md`.
- Agrega nuevas páginas creando archivos `.md`


5. Comparte:
   - la URL del repo
   - la URL pública del sitio

---

## Licencia

Define la licencia que usarás para el contenido (MIT, CC BY 4.0, etc.). Este template deja un footer con CC BY 4.0 como ejemplo.
